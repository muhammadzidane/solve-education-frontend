## Getting Started

First, install the depedencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

## Runing Local Server

Finally, run the development server

```bash
npm dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
